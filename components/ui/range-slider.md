# Cena Health — Range Slider Component Spec

---

## CSS Class Reference

| Class | Purpose |
|-------|---------|
| `.range-slider` | Styled native `input[type=range]` |

## Anatomy

```
input.range-slider[type="range"]
```

## Design Tokens

- Track height: 4px
- Track color: `var(--color-surface-secondary)`
- Track radius: `var(--radius-full)`
- Thumb size: 20px
- Thumb color: `var(--color-primary)`
- Thumb hover: `var(--color-primary-hover)`
- Focus ring: standard focus ring on thumb
- Disabled thumb: `var(--color-text-disabled)`
- Disabled track: `var(--color-surface-secondary)`
