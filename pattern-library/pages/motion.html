Motion

Duration scale and easing vocabulary. All interactive components derive their timing and curves from these tokens. Source: src/css/tokens-motion.css

Duration Scale

Click any row to play the animation.

--duration-normal 200ms Transitions

--duration-slow 400ms Major transitions

Easing Curves

Click each track to play the animation.

--ease-enter cubic-bezier(0.16, 1, 0.3, 1)

Objects arriving — spring deceleration, lands precisely

--ease-exit cubic-bezier(0.5, 0, 0.75, 0)

Objects leaving — accelerates quickly, exits clean

--ease-transition cubic-bezier(0.25, 0.1, 0.25, 1)

General-purpose — standard ease-in-out

--ease-emphasis cubic-bezier(0.34, 1.3, 0.64, 1)

Playful overshoot — slight bounce on arrival

--ease-spatial cubic-bezier(0.22, 0.68, 0, 1)

Navigation moves — spatial transitions between views

Motion Principles

  • Duration scales with distance and complexity
  • Enter animations use ease-enter (decelerate to rest)
  • Exit animations use ease-exit (accelerate from rest)
  • State changes use ease-transition
  • Always include prefers-reduced-motion: reduce overrides

Reduced Motion

Toggle to simulate prefers-reduced-motion: reduce. All duration demos above will play with 0ms duration.
