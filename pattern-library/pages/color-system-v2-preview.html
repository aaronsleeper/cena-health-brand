Cena Color System v2 — Preview

Runtime-interpolated via CSS color-mix(in oklch, …). Phase B1 review artifact for the 7 active families.
How to read this page:
  • Each family shows three anchor swatches (max / root / min) followed by the 11 computed stops (50, 100, 200, 300, 400, 500, 600, 700, 800, 900, 950).
  • Stops marked with ↑ logo below the cell correspond to a hex-exact logo identity color held in the separate logo-token namespace. The family palette stop at that position will drift slightly from the logo hex; that's expected per spec §1.5.
  • Interpolation is linear-in-L (k=1) — switched from cubic ease 2026-05-11 after preview feedback. Linear gives uniform perceptual stepping; cubic was clustering values at the extremes and starving the middle.
  • Each family also includes a "contrast pill" showing the canonical formula: fill 50 / border 200 (or 500 for sand lightness-compression) / heading text 500 / body text 800.
  • Final section: cross-family pill alignment — same pill formula across all 7 families, side-by-side, for visual parity check.
